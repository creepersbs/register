## Reporting Security Issues

If you discover a **security vulnerability** related to:

- Repository infrastructure
- Automation or scripts
- DNS management under CreeperHUB control

please report it responsibly.

### Preferred contact methods

-  Email: [`support@creeperhub.net`](mailto:support@creeperhub.net)
-  Contact form: https://creeperhub.net/contact  
  _(The form requires choosing either **Discord** or **Email** as a contact method.
  Discord requires joining the [official server.](https://creepers.sbs/discord))_

-  **High-risk or sensitive issues only**  
  You may contact the maintainer directly on GitHub - Discord:  
  [**`@giocoliere`**](`https://github.com/giocoliere`) - [**`@.giocoliere`**](`https://gioco.lol/discord`)

Please **do not publicly disclose** security issues before contacting us.

## Scope

This security policy applies **only** to:

- Infrastructure managed by CreeperHUB
- Repository configuration and automation

It does **not** apply to:

- User-managed DNS zones
- External Name Servers
- Content, services, or servers operated by subdomain owners
